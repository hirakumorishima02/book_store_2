My Book Store
====
我が家にある本を販売しているサイトです
今後、精力的に入荷予定かつ安価、そして丁寧に管理人（私）が梱包してお送りしています

## 導入したプラグイン・API・FWなど
- Stripe(決済システムプラグイン)
- Laravel5.8
- UIKit（CSSフレームワーク）
## デモサイト
https://old-book-store04.herokuapp.com
## 類似サイト
https://www.amazon.co.jp
BtoCではなく、CtoCで丁寧に梱包からお届けまで行うため、汚れた本などはなく、ほぼ新古書ばかりです。
心温かくなるような提供は大手通販サイトには真似できない要素です。
